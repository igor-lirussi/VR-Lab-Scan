# VR-Lab-Scan
## Description 
A Virtual Reality reconstruction of the laboratory, used as a base environment for some cognitive experiments, to reduce the bias between real and virtual experience.

### Topics:
- VR Virtual-Reality 
- 3D Scan

- Connect your device
- Run the app
